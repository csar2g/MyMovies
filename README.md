# My movies

# Descripción

Este proyecto consiste en el desarrollo de una página web sencilla enfocada en mostrar información sobre una película y uno de los actores que participa en ella.

La página incluye detalles relevantes como sinopsis, reparto, calificación, así como una sección dedicada a la biografía del actor seleccionado.

---

## Objetivo

El objetivo principal de esta actividad es aplicar conocimientos básicos de desarrollo web utilizando:

* HTML
* CSS
* Diseño responsivo con *media queries*

---

##  Características

*  Información general de la película
*  Calificación, director, presupuesto y otros datos
*  Sección de reparto (cast)
*  Página de información de un actor
*  Diseño responsivo adaptable a escritorio y dispositivos móviles

---

## Tecnologías utilizadas

* HTML5
* CSS3
* Flexbox
* Media Queries
